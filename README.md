# Matching Game

A simple card matching memory game built with vanilla JavaScript, HTML, and CSS.
The game demonstrates DOM manipulation, event handling, and basic game logic.

**Live Demo:** <https://mariaroim.com/matching-game/>  
**Repository:** https://github.com/mariaroim-dev/matching-game

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Features
- Randomly shuffles the game cards on start
- Flip cards to reveal symbols/images
- Check for matching pairs
- Cards stay revealed if matched
- If cards don’t match, they flip back after a short delay
- Game completes when all pairs are matched

## How to Play
1. Click on a card to reveal it.
2. Click on another card to try to find a pair.
3. If the cards match — they stay open.
4. If they don’t match — they flip back.
5. The game ends when all cards are matched.
